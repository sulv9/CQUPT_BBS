# CQUPT_BBS

（仅供学习，禁止用于商业用途）

## Applications

1. bbs - base
2. home - 主页
3. user - 用户系统
4. write - 编辑

### bbs - base

左边侧边栏left_sidebar：顶部图标，中间两个导航图标，底部用户头像（未登录则不显示头像）。

中间主体内容content。

右侧栏right_sidebar：其中右侧栏有两套，一套是主页/搜索，一套是文章/作者页。

### home - 主页

顶部未登录则提示去登录，已登录则显示Follow People；下面有两列Following和All显示帖子，文章用分割线隔离。
